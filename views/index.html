Welcome to Movie Library

Movie Library is a secure web application that allows authenticated users to manage their movie collection. Create an account, log in, and start adding your favorite movies to your personal library.

Key Features

  • User Authentication: Secure login and registration with password hashing (bcrypt)
  • Session Management: Session-based authentication with MongoDB session storage
  • CRUD Operations: Create, Read, Update, and Delete movies through Web UI
  • Realistic Data: 23 movies with 7 meaningful fields (title, genre, director, year, rating, duration, description)
  • Security: HttpOnly cookies, authorization middleware, secure error handling
  • Professional UI: Responsive design with modern styling

Getting Started

To manage your movie collection, you need to:

  1. Create an Account
  2. Log In
  3. Go to Movies

Security Features

  • Password Hashing: All passwords are hashed using bcrypt algorithm
  • Authentication Middleware: Protects write operations (CREATE, UPDATE, DELETE)
  • Session Security: HttpOnly flag prevents JavaScript access to cookies
  • Input Validation: All user inputs are validated on server and client
  • Error Handling: Generic error messages to prevent information leakage
  • Authorization: Users can only modify their own movies

Technical Stack

  • Backend: Node.js + Express.js
  • Database: MongoDB
  • Authentication: express-session + bcrypt
  • Frontend: HTML5 + CSS3 + JavaScript (ES6)
  • Session Storage: MongoDB (connect-mongo)
